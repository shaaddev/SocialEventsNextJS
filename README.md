# Social Events

__This is a full stack project created using _NextJS_ and _MongoDB_.__

_Originally created using the python based web framework Django_

## Getting Started

Clone project
```zsh
$ git clone <insert_here>
```

Install dependencies
```zsh
$ nvm use 20.9.0

$ npm install
```

Run in Development
```zsh
$ npm run dev
```
